# InsightHub
📊 InsightHub - A modern Indian Stock Market Dashboard built with React 19, TypeScript, Material UI, and Redux Toolkit. Features real-time stock tracking, watchlist, portfolio management, and market screener with Twelve Data API integration.


🚀 InsightHub - Indian Stock Market Dashboard

A professional SaaS-style admin dashboard for tracking Indian stocks (NIFTY 50, SENSEX).

Tech Stack:
• React 19 + TypeScript + Vite
• Material UI v7
• Redux Toolkit
• React Router v7
• Recharts
• Supabase Auth
• Twelve Data API

Features:
✅ Market Overview - 50 Indian stocks with live prices
✅ Company Insights - Detailed analytics & charts
✅ Watchlist - Track favorite stocks
✅ Portfolio - Holdings tracker with P&L
✅ Screener - Top gainers/losers with filters
✅ Dark/Light Theme
✅ Responsive Design
